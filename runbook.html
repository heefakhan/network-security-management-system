üîß Troubleshooting Guide

‚ùå Backend Not Responding

Symptom: Dashboard shows "Backend not running"
1. Check if backend terminal is open
2. Verify port 8001 is free:
netstat -ano | findstr :8001
3. Kill process if needed:
taskkill /PID [number] /F
4. Restart backend:
uvicorn main:app --reload --port 8001

üìä Graphs Not Showing

Symptom: Blank graphs or no data
1. Click "GRAPH ON" button
2. Check Chart.js CDN:
F12 ‚Üí Network tab ‚Üí chart.js loaded?
3. Clear browser cache
4. Refresh page (F5)

üö´ Block/Unblock Failing

Symptom: Buttons not working
1. Check backend connection
2. Verify API endpoint:
curl http://localhost:8001/api/devices
3. Check CORS in main.py
4. Restart backend

üíæ Database Issues

Symptom: No data or stale data
1. Check if network.db exists
2. Verify database permissions
3. Restore from backup:
copy backend\network_backup.db backend\network.db
4. Restart backend

üåê Live Server Issues

Symptom: HTML not loading properly
1. Install Live Server extension
2. Right-click ‚Üí Open with Live Server
3. Check port 5500 is free
4. Or open directly:
file:///C:/path/to/dashboard.html
